## 安装后端

- 进入此目录，然后安装依赖

``` bash
npm install
```

- 或者使用 `yarn` 安装

```bash
yarn
```
## 初始化数据库（添加测试数据）

- 启动initDB.js

```
node initDB.js

```
- 然后终止这个程序

## 启动后端

``` bash
# serve with hot reload at localhost:3000
npm run start
```

## 安装前端

- 进入client目录，然后安装依赖

``` bash
npm install
```

- 或者使用 `yarn` 安装

```bash
yarn
```

## 启动前端

``` bash
# serve with hot reload at localhost:8080
npm run dev
```
## 访问

浏览器访问localhost:8080即可看到页面
